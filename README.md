# weather-app

A basic weather app built with vue.js using the OpenWeatherMap API (https://openweathermap.org) 

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
